Digidiary

Technology used: HTML | CSS | JavaScript

Digidiary is a personal project that acts like a digital diary without the hassle of turning pages and buying a new one for a new year. Users can not only log their daily thoughts/activities but also make a to-do list and set reminders for events. They can choose from yearly events (eg. birthdays), monthly events (eg. rent date), and one-time events (eg. a concert) and get reminders for them accordingly. As a student, with multiple assignments and exam dates to remember, this project helps a lot!
